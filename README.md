This is Cityscape, an IOS application built in SwiftUI that shows the best events going on in your city at any given time.

This app was built as a final project for Intro to Programming Using Swift for IOS App Development at Boston College in Fall 2026

This app extensively uses Google Firebase and the Google Maps SDK. Firebase packages include:
- FirebaseAuth
- FirebaseFirestore
- FirebaseStorage
